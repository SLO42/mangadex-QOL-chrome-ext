MangaDex QOL — Icon Generator

Generates icon16.png, icon32.png, icon48.png, and icon128.png from the embedded SVG. Save them next to this HTML file (in the icons/ folder) before publishing to the Chrome Web Store.

Why a generator? Why not commit PNGs?

Binary assets in git are fine for small icons, but checking them in means they have to stay in sync with the SVG. The generator makes the SVG the single source of truth and produces fresh PNGs on demand. For unpacked development installs Chrome doesn't actually need the PNGs — it just falls back to the default puzzle-piece icon and prints a warning.
